# Imágenes de Kits

Coloca aquí las imágenes de los kits con los siguientes nombres:

- `kit1.jpg` - KIT 1 ENVASADORA 3.6 KG
- `kit2.jpg` - KIT 2 ENVASADORA 2 KG
- `kit3.jpg` - KIT 3 ENVASADORA TARRINAS
- `kit4.jpg` - KIT 4 CAJA COMUN

Las imágenes se mostrarán automáticamente en el formulario de registro de herramientas.

Formato recomendado:
- Formato: JPG o PNG
- Tamaño: Mínimo 300x300px, recomendado 600x600px
- Peso: Optimizado para web (máximo 500KB por imagen)
